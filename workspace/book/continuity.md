# Book Continuity Tracker

This file starts empty. The writer agent updates it after every chapter. The author can also edit it directly to plant things the writer should remember.

## Metaphors used

(One line per metaphor, with the chapter it appeared in. Never reuse. If a metaphor shows up twice, the second use gets cut.)

## Anecdotes deployed

(Same. Each story listed once with its chapter. If you're about to use a story you've already used, pick a different story.)

## Forward references

(Things the text promises to return to later.
Format: "Chapter X mentioned [topic]. Must be addressed by Chapter Y.")

## Callbacks paid off

(Forward references that have been resolved. Move them here from the Forward references section when you pay them off.)

## Character / concept appearances

(Track recurring characters, tools, or concepts.
Which chapter introduced them. Which chapters reference them.)

## Tone notes

(Any observations about drift, inconsistencies, or voice issues. The weekly review writes into this.)
